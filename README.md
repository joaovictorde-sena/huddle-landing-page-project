# Huddle Landing Page

Uma landing page simples e responsiva que apresenta o conceito do Huddle, uma plataforma para construir comunidades envolventes.

---

## 📋 Estrutura do Projeto

O projeto utiliza HTML5 e CSS3 para criar uma interface visualmente atraente e funcional. A página é composta por:

- **Header:** Com o logotipo do Huddle.
- **Main:** Inclui:
  - Uma ilustração de destaque.
  - Um título e descrição do serviço.
  - Um botão para registro.
- **Footer:** Links para redes sociais (Facebook, Twitter, Instagram).

---

## 🛠️ Tecnologias e Recursos

- **HTML5:** Estrutura do documento.
- **CSS3:** 
  - `style.css`: Estilização principal.
  - `reset.css`: Reset de estilos padrão.
  - `responsive.css`: Estilos para dispositivos móveis.
- **Google Fonts:** Fontes `Open Sans` e `Public Sans`.
- **Font Awesome:** Ícones das redes sociais.

---

## 📱 Responsividade
O layout é otimizado para dispositivos móveis, garantindo uma boa experiência em diferentes tamanhos de tela.
